# devops-netology-2026

Инструкция:

1. Добавить файл в ГИТ через git add
2. Отредактировать через редактор (например, nano) файл и сохранить
3. Для правильного обновления ввести команду git add
4. Закоммитить новый файл git commit / -a. В открывшемся окне можно написать написать номер версии и т.д.
5. Передать новый файл в ГИТ ХАБ командой файла git push
