# devops-netology-2026

                                Инструкция:

1. Для работы в ГИТ сначала нужно иницил. каталог командой git init
2. Добавить файл в ГИТ через git add
3. Отредактировать через редактор (например, nano) файл и сохранить
4. Для правильного обновления ввести команду git add
5. Закоммитить новый файл git commit / -a. В открывшемся окне можно написать написать номер версии и т.д.
6. Передать новый файл в ГИТ ХАБ командой файла git push
_________________________________________________________

1. Для повторной работы в ГИТ нужно иницил. каталог командой git init 
2. Для повторной работы в ГИТ нужно повторно зайти в рабочий каталог cd...
3. Открыть файл для редактирования и отредактировать через
   редактор (например, nano) файл и сохранить
4. Для правильного обновления ввести команду git add
5. Закоммитить новый файл git commit / -a. 
   В открывшемся окне можно написать написать номер версии и т.д.
6. Передать новый файл в ГИТ ХАБ командой файла git push
